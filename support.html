常見問題與使用說明

快速開始

1

取得 Gemini API Key

前往 aistudio.google.com，用 Google 帳號登入後點擊「Get API Key」，即可免費申請。每日提供 250 次免費配額，隔天自動重置。

2

在 EquaClip 設定中填入 API Key

點擊系統列的 圖示，選擇「設定」，將剛才取得的 API Key 貼入欄位並儲存。Key 僅儲存在你的裝置本地，不會上傳至任何伺服器。

3

開啟輔助使用權限

EquaClip 需要輔助使用權限才能監聽全域快捷鍵。
路徑：系統設定 → 隱私權與安全性 → 輔助使用 → 開啟 EquaClip 的開關

Go to: System Settings → Privacy & Security → Accessibility → enable EquaClip

可能原因：

  • API Key 錯誤 — 進入設定確認 Key 是否貼入完整
  • 網路問題 — 確認裝置能連線至 Google 服務
  • 免費額度用完 — 每日 250 次，隔天重置後即可繼續使用

Possible causes:

  • Invalid API Key — Go to Settings and verify the key is pasted correctly
  • Network issue — Make sure your device can reach Google services
  • Daily quota exceeded — Free tier is 250 requests/day; resets at midnight

以下方式可提升準確率：

  • 截圖範圍盡量貼近公式，減少多餘空白
  • 確保螢幕顯示清晰，縮放比例正常
  • 印刷體（電子書、PDF）比手寫準確率高

Tips to improve accuracy:

  • Crop tightly around the formula to minimize surrounding whitespace
  • Ensure the screen is sharp and at normal zoom level
  • Printed text (eBooks, PDFs) gives better results than handwriting
