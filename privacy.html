概述

EquaClip（以下簡稱「本應用程式」）是一款執行於你個人 Mac 電腦上的工具程式。我們非常重視你的隱私，並以「最小資料原則」設計本應用程式——我們不運營任何伺服器，不收集、不儲存、不傳輸任何個人資料。

本政策說明本應用程式如何運作，以及你的資料在何處流動。

EquaClip is a utility application that runs entirely on your personal Mac. We take your privacy seriously and designed this app on a "minimal data" principle — we operate no servers and collect, store, or transmit no personal data whatsoever.

This policy explains how the app works and where your data goes.

我們收集哪些資料

我們不收集任何個人資料。

本應用程式不會：

  • 追蹤你的使用行為或統計資料
  • 收集裝置識別碼、IP 位址或位置資訊
  • 上傳截圖至我們的伺服器（因為我們沒有伺服器）
  • 建立使用者帳號或個人檔案

本地儲存的資料

以下資料儲存在你的裝置本地，僅供本應用程式使用：

  • Gemini API Key：加密儲存於 macOS Keychain，不會離開你的裝置
  • 應用程式偏好設定：快捷鍵設定等，儲存於本機 UserDefaults

We collect no personal data.

The app does not:

  • Track your usage behavior or analytics
  • Collect device identifiers, IP addresses, or location
  • Upload screenshots to our servers (we have none)
  • Create user accounts or profiles

Data stored locally

The following data is stored only on your device for the app's use:

  • Gemini API Key: Encrypted in macOS Keychain and never leaves your device
  • App preferences: Shortcut settings, etc., stored in local UserDefaults

第三方服務：Google Gemini API

當你使用 EquaClip 辨識公式時，截圖圖像會直接從你的裝置傳送至 Google Gemini API 進行辨識。這是唯一的外部資料傳輸。

請注意：

  • 傳送的內容僅包含截圖圖像，不含任何個人識別資訊
  • API Key 透過 HTTPS request header（x-goog-api-key）傳輸，不會出現在 URL 中，降低 Key 外洩風險
  • 截圖以隨機檔名暫存，辨識流程結束後自動刪除，不永久保留
  • Google 對此資料的處理方式依循其自身的隱私權政策
  • 你可前往 Google 隱私權政策 查閱詳情
  • API Key 由你自行申請並管理，EquaClip 不持有或存取你的 Google 帳號

When you use EquaClip to recognize a formula, the screenshot is sent directly from your device to the Google Gemini API. This is the only external data transfer.

Please note:

  • Only the screenshot image is transmitted — no personal identifiers
  • Your API Key is sent via an HTTPS request header (x-goog-api-key), never in the URL, reducing exposure in logs or proxies
  • Screenshots are saved with a random filename and automatically deleted after recognition — never stored permanently
  • Google's handling of this data is governed by their own privacy policy
  • See Google's Privacy Policy for details
  • The API Key is managed entirely by you; EquaClip does not access your Google account

安全性設計

🔒 Security by Design
API Key 儲存：macOS Keychain
Gemini API Key 加密儲存於 macOS Keychain，而非一般的 UserDefaults 或設定檔。若你曾使用舊版，啟動時會自動將舊值搬移至 Keychain 並清除原始紀錄。App 不會將 API Key 寫入專案檔、設定檔或任何雲端同步文件中。
API Key 傳輸：HTTPS Header，不放 URL
呼叫 Gemini API 時，API Key 透過 HTTPS request header x-goog-api-key 傳遞，不會出現在 URL query string 中。這能有效降低 Key 被記錄在 proxy log、瀏覽器歷史、診斷資訊或錯誤訊息中的風險。
截圖暫存：隨機檔名 + 辨識後自動刪除
使用者框選的截圖會以隨機產生的檔名暫存，辨識流程結束後立即刪除。App 本身不永久保存任何截圖內容。
錯誤訊息保護：不暴露原始 API 回應
API 錯誤發生時，使用者只會看到簡化後的提示（例如「API Key 無效」、「網路錯誤」），不會顯示 Gemini 的 raw response 內容。詳細的錯誤資訊僅在 Debug 模式下輸出，避免敏感技術資訊外洩。
🔒 Security by Design
API Key Storage: macOS Keychain
Your Gemini API Key is encrypted and stored in macOS Keychain — not in UserDefaults or any config file. If you used an older version, the app automatically migrates your key to Keychain on launch and removes the old value. The key is never written to project files, config files, or cloud-synced documents.
API Key Transmission: HTTPS Header, Not URL
When calling the Gemini API, the API Key is passed via the HTTPS request header x-goog-api-key — never in the URL query string. This prevents the key from appearing in proxy logs, browser history, diagnostic reports, or error messages.
Screenshot Handling: Random Filename + Auto-Deleted
Screenshots are temporarily saved with a randomly generated filename and deleted immediately after the recognition process completes. The app never stores screenshot content permanently.
Error Message Protection: No Raw API Responses
When an API error occurs, users only see simplified messages (e.g. "Invalid API Key", "Network error") — the raw Gemini response is never displayed. Detailed error output is only available in Debug mode, preventing sensitive technical information from being exposed.

隱私控制設定

EquaClip 提供多項隱私相關設定，讓你自行決定資料的可見範圍：

通知中的 LaTeX 預覽
可在設定中關閉辨識完成通知內的 LaTeX 內容預覽，避免公式出現在通知中心或鎖定螢幕上。
選單列最近辨識結果
可關閉選單列顯示最近 3 次辨識結果的功能。關閉後，App 不會在選單列保留任何辨識歷史，結果只會複製到剪貼板。
權限狀態一覽
設定頁顯示輔助使用、螢幕錄製、通知三項權限的目前狀態，並提供一鍵跳轉至 macOS 系統設定的入口。

EquaClip offers several privacy settings, giving you control over what is visible and retained:

LaTeX Preview in Notifications
You can disable the LaTeX content preview in recognition notifications, preventing formula content from appearing in Notification Center or on your lock screen.
Recent Results in Menu Bar
You can disable the last 3 recognition results shown in the menu bar. When disabled, the app retains no recognition history — results are only copied to the clipboard.
Permission Status at a Glance
The Settings page shows the current status of Accessibility, Screen Recording, and Notifications permissions, with a one-tap shortcut to open the relevant macOS System Settings pane.

資料安全

由於 EquaClip 不收集任何資料，也不運營任何伺服器，因此不存在資料外洩或伺服器端資料被盜的風險。

你的 Gemini API Key 使用 macOS Keychain 機制加密儲存，這是 Apple 提供的最高等級本地安全儲存方式，其他應用程式無法讀取。

Because EquaClip collects no data and operates no servers, there is no risk of a server-side data breach.

Your Gemini API Key is encrypted and stored in macOS Keychain — Apple's highest-level local secure storage, inaccessible to other applications.
