概述

MathSnap（以下簡稱「本應用程式」）是一款執行於你個人 Mac 電腦上的工具程式。我們非常重視你的隱私，並以「最小資料原則」設計本應用程式——我們不運營任何伺服器，不收集、不儲存、不傳輸任何個人資料。

本政策說明本應用程式如何運作，以及你的資料在何處流動。

MathSnap is a utility application that runs entirely on your personal Mac. We take your privacy seriously and designed this app on a "minimal data" principle — we operate no servers and collect, store, or transmit no personal data whatsoever.

This policy explains how the app works and where your data goes.

我們收集哪些資料

我們不收集任何個人資料。

本應用程式不會：

  • 追蹤你的使用行為或統計資料
  • 收集裝置識別碼、IP 位址或位置資訊
  • 上傳截圖至我們的伺服器（因為我們沒有伺服器）
  • 建立使用者帳號或個人檔案

本地儲存的資料

以下資料儲存在你的裝置本地，僅供本應用程式使用：

  • Gemini API Key：加密儲存於 macOS Keychain，不會離開你的裝置
  • 應用程式偏好設定：快捷鍵設定等，儲存於本機 UserDefaults

We collect no personal data.

The app does not:

  • Track your usage behavior or analytics
  • Collect device identifiers, IP addresses, or location
  • Upload screenshots to our servers (we have none)
  • Create user accounts or profiles

Data stored locally

The following data is stored only on your device for the app's use:

  • Gemini API Key: Encrypted in macOS Keychain and never leaves your device
  • App preferences: Shortcut settings, etc., stored in local UserDefaults

第三方服務：Google Gemini API

當你使用 MathSnap 辨識公式時，截圖圖像會直接從你的裝置傳送至 Google Gemini API 進行辨識。這是唯一的外部資料傳輸。

請注意：

  • 傳送的內容僅包含截圖圖像，不含任何個人識別資訊
  • Google 對此資料的處理方式依循其自身的隱私權政策
  • 你可前往 Google 隱私權政策 查閱詳情
  • API Key 由你自行申請並管理，MathSnap 不持有或存取你的 Google 帳號

When you use MathSnap to recognize a formula, the screenshot is sent directly from your device to the Google Gemini API. This is the only external data transfer.

Please note:

  • Only the screenshot image is transmitted — no personal identifiers
  • Google's handling of this data is governed by their own privacy policy
  • See Google's Privacy Policy for details
  • The API Key is managed entirely by you; MathSnap does not access your Google account

系統權限說明

MathSnap 需要以下 macOS 系統權限才能正常運作：

權限用途是否傳送至外部
輔助使用 監聽全域快捷鍵（⌘⇧L），讓你在任何 App 中都能觸發截圖 否
螢幕錄製 截取你框選區域的螢幕畫面，用於公式辨識 截圖傳送至 Google Gemini API

你可以隨時在「系統設定 → 隱私權與安全性」中撤銷上述權限。

MathSnap requires the following macOS system permissions to function:

PermissionPurposeData sent externally?
Accessibility Listens for the global shortcut (⌘⇧1) so you can trigger capture from any app No
Screen Recording Captures the screen region you select for formula recognition Screenshot sent to Google Gemini API

You can revoke these permissions at any time in System Settings → Privacy & Security.

資料安全

由於 MathSnap 不收集任何資料，也不運營任何伺服器，因此不存在資料外洩或伺服器端資料被盜的風險。

你的 Gemini API Key 使用 macOS Keychain 機制加密儲存，這是 Apple 提供的最高等級本地安全儲存方式，其他應用程式無法讀取。

Because MathSnap collects no data and operates no servers, there is no risk of a server-side data breach.

Your Gemini API Key is encrypted and stored in macOS Keychain — Apple's highest-level local secure storage, inaccessible to other applications.

政策變更

若本隱私權政策有重大變更，我們會在新版本的應用程式中告知使用者，並更新本頁面頂端的「最後更新」日期。

繼續使用本應用程式即表示你同意最新版本的隱私權政策。

If we make significant changes to this policy, we will notify users within the app and update the "Last updated" date at the top of this page.

Continued use of the app constitutes acceptance of the updated policy.
