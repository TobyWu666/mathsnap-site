如何取得 Gemini API Key

Gemini API Key 完全免費，每天 250 次配額，按照以下步驟申請。

1
前往 Google AI Studio

在瀏覽器開啟 aistudio.google.com，這是 Google 提供的免費 AI 開發平台。

登入 Google 帳號後，在左側導覽列找到並點擊「Get API key」選項。

點擊系統列的 圖示，開啟「設定」，將複製的 API Key 貼入欄位並儲存。
